# 🚀 Projeto: Upper Billing

## 🎯 Objetivo

Sistema SaaS para emissão e gestão de cobranças.

---

## 🏗 Stack

Backend:
- Node.js
- TypeScript
- Serverless v4
- DDD
- Clean Architecture

Frontend:
- Next.js
- Tailwind
- Shadcn

Infra:
- AWS
- DynamoDB
- S3
- OpenTelemetry
- GitHub Actions CI/CD

---

## 🧠 Domínios

- Billing
- Subscription
- Invoice
- Payment

---

## 📦 MVP

- Criar cliente
- Criar cobrança
- Listar cobranças
- Webhook de pagamento

---

## 🔐 Requisitos Não Funcionais

- Multi-tenant
- Observabilidade com OpenTelemetry
- Logs estruturados
- Idempotência
- Testes unitários obrigatórios

---

## 🚀 Deploy

- develop → ambiente dev
- main → produção
- Versionamento automático via Conventional Commits
